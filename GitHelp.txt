Команды для регистрации моих данных:

git config --global user.email "ayelitaaxn@gmail.com"

Команды только для одного проекта:
git init            | Инициализация Гита
git remote add origin https://github.com/lukaaxn/OOP-60-2.git | Добавление репозитория

git status | Статус репозитория


git add .           | Добавление файлов в буфер обмена
git commit -m "message"  | Сохранение всего из буфера обмена
git push origin master   | Отправка изменений в репозиторий
